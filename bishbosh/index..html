Bish-Bosh 2.0







Explanation
User enters a number in the first three text boxes.First is the bish number ,second Bosh and finally the limit.Result will show in a div as numbers from 1 to the limit, but some exceptions. If a number is divisible by both bish number and bosh number, then print Bish Bosh.If the number is only divisible by bish number,then print Bish.If the number is not divisible by both bish number and bosh number, then print the number itself.
